HTML Elements Definitions

<h1>
This tag is used for a heading on your page. This is the largest heading.
<h2>
This tag is used for a heading on your page. This is the second largest heading.
<h3>
This tag is used for a heading on your page. This is the third largest heading.
<h4>
This tag is used for a heading on your page. This is the fourth largest heading.
<h5>
This tag is used for a heading on your page. This is the fifth largest heading.
<h6>
This tag is used for a heading on your page. This is the sixth largest heading.
<p>
This tag creates a paragraph on your page. This is useful for keeping text separate.
<b>
This tag lets the user set text as bold. This is useful for emphasizing certain text.
<i>
This tag lets the user set text as italic. This is useful for emphasizing certain text.
<sup>
This tag lets the user set text as superscript. This is useful for dates like 4th.
<sub>
This tag lets the user set text as subscript. This is useful for chemical makeup like CO2.
<br>
This tag lets the user create a line break. This is useful for formatting.
<hr>
This tag lets the user create a horizontal line on the page. This is useful for formatting and organization.
<strong>
This tag lets the user emphasize a certain word or object. By default, this will put the item in bold
<em>
This tag lets the user indicate subtle emphasis. By default, this will put the item in italics
<blockquote>
This tag lets the user set a text entry as a quote. This is usually used for large quotes
<q>
This tag lets the user set a text entry as a quote. This is typically used for shorter quotes.

<cite>
This tag lets the user to cite a reference source. This is typically not meant to be used for a person's name in HTML 5.
<dfn>
This tag lets the user define a new term on the page. Safari and chrome do not set the term in italics with this tag.
